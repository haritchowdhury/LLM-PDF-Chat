# NextJS 15 Authentication Tutorial

Simple implementation of Auth.js with Prisma in Next.js 15

## Requirements

- Node.js 22+

## Quick Start

1. **Clone the repository**

```bash
# For tutorial
git clone -b starter

# For complete code
git clone
```

2. **Install & Run**

```bash
npm install
npm run db:migrate
npm run dev
```

## Tech Stack

- Next.js 15
- Node.js 22
- TypeScript
- Prisma
- Auth.js
- Langchain
- Upstash
